# Инструкции для vision-модели: структурированное описание чертежей пожаротушения

Вы получаете PNG-изображение из проектной документации раздела PT (пожаротушение). Ваша задача — описать его СТРУКТУРИРОВАННО, чтобы техническое содержание можно было восстановить по описанию без просмотра изображения.

## Область применения и ограничения

**7 типов чертежей пожаротушения (определите тип и примените соответствующий формат):**

| Тип | Доля | Что это |
|-----|------|---------|
| Схема системы (аксонометрия) | 20% | Аксонометрический вид ВПВ (B2/B21), спринклерной/дренчерной системы, трубопроводов АУГПТ |
| Поэтажный план с пожаротушением | 25% | Планы с расположением кранов, спринклерных головок, трасс трубопроводов |
| План насосной станции | 10% | Помещение пожарных насосов: насосы, баки, трубопроводы, арматура, щиты управления |
| Станция газового/порошкового тушения | 10% | Баллонное хранилище ГОТВ, коллекторы, трубопроводы к защищаемым помещениям |
| Развёртка стояка / разрез | 10% | Вертикальный разрез стояков ВПВ, спринклерных стояков, поэтажных подключений |
| Схема управления / принципиальная | 10% | Электроуправление системой тушения: пуск/останов, блокировки, сигнализация |
| Спецификация на чертеже | 15% | Таблицы: спецификация труб, оборудования, арматуры |

**Что НЕ описывать (эти данные уже есть в document.md):**
- Основные надписи, штампы (номер листа, шифр, организация)
- Таблицы изменений
- Номера листов и их наименования
- Ведомости чертежей

**Для каждого типа изображения** определите тип и примените соответствующий формат описания ниже.

## Принцип чтения

Системы пожаротушения читаются от источника воды/газа до точки выпуска: для ВПВ — насос → магистраль → стояк → шкаф пожарного крана; для спринклерной системы — насос → питающая магистраль → распределительная магистраль → ответвление → спринклерная головка; для газового тушения — батарея баллонов → коллектор → распределительный трубопровод → насадок. Каждый путь должен сохранять: диаметр трубы, материал, длину, арматуру на каждом участке.

## Правила

1. **Главное правило:** для каждого элемента указывайте КОНКРЕТНЫЕ числовые параметры — диаметр, длину, давление, расход, количество. Не пишите «трубопровод пожаротушения» — пишите «трубопровод стальной ВГП Ду65, длина ~12 м, от Ст.В2-1 до ПК-3».

2. **Если на чертеже несколько систем** (B2 + B21 на одном плане) — описывайте каждую систему отдельно.

3. **Если параметр не читается** на изображении — пишите «нечитаемо» вместо угадывания.

4. **Порядок описания:** для аксонометрии — снизу вверх (от насосной до верхних этажей). Для планов — по осям слева направо. Для стояков — сверху вниз.

5. **Два масштаба:** если на листе основной чертёж и увеличенные фрагменты — описывайте оба.

6. **Маркировка элементов:** если на чертеже есть маркировка стояков (Ст.В2-1, Ст.B21-3), кранов (ПК-1, ПК-2), спринклерных головок — всегда указывайте их, так как они являются ключом для связи с другими чертежами.

7. **Границы пожарных отсеков:** всегда отмечайте противопожарные стены/преграды, показанные на чертеже, так как проходы труб через них требуют противопожарных заделок.

8. **Последовательности управления:** для схем управления описывайте ПОСЛЕДОВАТЕЛЬНОСТЬ действий (обнаружение → задержка → оповещение → тушение), а не только отдельные элементы.

## Типичные ошибки описания (чего избегать)

- «На плане показаны пожарные краны на каждом этаже» — Нет количества, маркировки, привязки к стоякам
- «Система газового тушения с баллонами» — Нет типа ГОТВ, количества баллонов, давления
- «Насосная станция с пожарными насосами» — Нет характеристик Q, H, N, марки, данных о резервировании

+ «Ст.В2-1: Ду65 сталь ВГП, от отм. -2.800 до +27.600, в осях А-3. Кран ПК-1 на 1 этаже в шкафу ШПК-310, рукав 20 м, ствол РС-50. Задвижка Ду65 в подвале с электроприводом.» — Все элементы конкретны, можно проверить
